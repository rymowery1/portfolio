# RyMow.com — Information Architecture

---

## Site Goal
Convert visitors into clients or collaborators by communicating Ryan's design expertise, aesthetic POV, and personality.

## Primary User Action
→ Get in touch / Start a project

---

## Pages & Sections

### 1. Home
The first impression. Establishes brand voice and directs traffic.

- **Hero** — Name, short positioning line (who you are + what you do)
- **Selected Work** — 3–4 featured projects (teaser, not exhaustive)
- **About Teaser** — 1–2 sentences + link to full About
- **CTA** — "Let's work together" → links to Contact

---

### 2. Work (Portfolio)
The main trust-builder. Where clients decide if you're the right fit.

- **Project Grid** — All projects, filterable by type (Branding, UI/UX, Print, etc.)
- **Each Project Card** — Thumbnail, title, category, year

#### 2a. Project Detail Page (one per project)
- Hero image
- Project overview (client, role, year)
- Problem / Approach / Outcome
- Image gallery / process shots
- Next project → (keeps people browsing)

---

### 3. About
Where people decide if they like *you*, not just your work.

- **Photo** — approachable, on-brand
- **Bio** — who you are, your design philosophy, what drives you
- **Skills / Services** — what you actually offer (branding, UI, etc.)
- **Clients / Logos** — social proof (if applicable)
- **CTA** — → Contact

---

### 4. Contact
Remove all friction to reaching out.

- **Short intro line** — warm, inviting
- **Contact form** — Name, Email, Message (simple)
- **Alt contact** — email address directly, social links
- **Response expectation** — "I'll get back to you within 48 hours"

---

## Global Elements (every page)

| Element | Content |
|---|---|
| **Nav** | Logo/Name · Work · About · Contact |
| **Footer** | © Ryan Mow · Social links · Email |

---

## Content Types

| Type | Lives In |
|---|---|
| Project case studies | Work → Project Detail |
| Bio + philosophy | About |
| Skills / services | About |
| Testimonials (optional) | About or Home |
| Process / behind the scenes | Project Detail |
